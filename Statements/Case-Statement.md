# Case Statement - Switch

```cs
switch (stateAppr)
            {
                case "WI":
                    stateFull = "Wisconsin";
                    break;
                case "AZ":
                    stateFull = "Arizona";
                    break;
                case "IA":
                    stateFull = "Iowa";
                    break;
                case "ID":
                    stateFull = "Idaho";
                    break;
                case "IL":
                    stateFull = "Illinios";
                    break;
                case "MI":
                    stateFull = "Michigan";
                    break;
                case "MN":
                    stateFull = "Minnesota";
                    break;
                case "NV":
                    stateFull = "Nevada";
                    break;
                case "UT":
                    stateFull = "Utah";
                    break;
                case "WY":
                    stateFull = "Wyoming";
                    break;
                default:
                    stateFull = "Unknown";
                    break;
            }
```
